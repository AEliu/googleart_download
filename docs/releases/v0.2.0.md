# v0.2.0

Second public release of `googleart-download`.

## Highlights

- Add configurable JPEG output quality:
  - `--jpeg-quality`
  - `--jpeg-preset web|balanced|archive`
- Add a Chinese README entry page (`docs/README.zh-CN.md`).
- Replace README mock-style TUI visuals with generated snapshots based on the current TUI renderer.
- Refine TUI and summary wording:
  - clearer skipped-artwork display
  - more accurate `Artwork` summary labeling

## Notes

- JPEG quality controls only affect JPEG outputs.
- Very large artworks still follow the safer TIFF/BigTIFF path by default.
- README screenshots are generated assets and may still need future presentation tuning for web and mobile layouts.
